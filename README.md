# Cosmic Orrery

A beautiful Three.js-powered universe and solar-system simulation with accelerated adjustable time, realistic relative orbital periods, bloom lighting, stars, rings, moons, and a modern glassmorphism HUD.

## Run locally

```bash
npm run build
npx serve dist
```

The app imports Three.js modules from ESM CDN at runtime so the repository stays lightweight and deploys as a static site.
